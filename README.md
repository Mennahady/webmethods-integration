# webmethods-integration
The project demonstrates basic enterprise integration concepts including:
- Input validation
- Service orchestration
- REST API consumption
- Conditional routing
- Messaging (queues)
- Error handling

## Integration Scenarios
### Flow 1
- Validates user input (National ID length, required fields)
- Routes requests based on Age
- Invokes REST services using HTTP GET

### Flow 2
- Receives output from Flow 1
- Performs additional processing
- Demonstrates service chaining and response handling

## Technologies
- webMethods Designer
- REST APIs
- Messaging Queues
